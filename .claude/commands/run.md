Start the Spendly development server.

Steps:
1. Check that `database/spendly.db` exists. If it doesn't, run `venv/bin/python database/db.py` first to initialise and seed the database.
2. Start the server with `venv/bin/python app.py`.
3. Tell the user the app is running at http://127.0.0.1:5001 and remind them of the sample credentials: `nitish@example.com` / `password123`.
